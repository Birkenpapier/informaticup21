Hier eine kleine Anleitung zur Installation für Docker:

1. Docker für euer jeweiliges Bertiebssystem herunterladen und installieren:
    https://docs.docker.com/compose/install/

2. Kommandozeile in diesem Ordner öffnen bzw. hinnavigieren.

3. Eingabe:     docker build -t informaticup-gruppe7 .
                docker run informaticup-gruppe7

4. Erwartete Ausgabe: TESTAUSGABE IM DOCKER

Gleiches könnt ihr dann auch mit dem Beispieldocker im Informaticup-Repo ausprobieren. Nicht wundern,
wenn am Anfang nichts passiert. Der !!!Server wartet eine gewisse Zeit bevor ein Spiel gestartet wird!!!, damit weitere
Spieler beitreten können.
